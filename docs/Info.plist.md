# Info plist

## File picker

Following configs are required by file_picker.

```xml
<key>UISupportsDocumentBrowser</key>
<true/>

<key>LSSupportsOpeningDocumentsInPlace</key>
<true/>
```

- https://pub.dev/packages/file_picker
